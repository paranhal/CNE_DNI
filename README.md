# CNE_DNI

충남/대전 지역 학교 유무선 데이터 진단 및 개선 사업 관련
크롤링/정제/분석 작업 프로젝트

## Environment
- Python 3.13
- Virtual Env: .venv
- Main libs: pandas, openpyxl, jupyter

## Start
source .venv/bin/activate
jupyter lab

## Structure
- src/ : Python scripts
- notebooks/ : Jupyter notebooks
- config/ : local/shared path configs
- output/ : outputs (git ignored)
- logs/ : logs (git ignored)

# 사전 조사 (충남 전체 폴더·모든 시트 스캔)

충남 719개 학교 자료 위치 파악을 위해 **모든 폴더**와 **모든 엑셀 파일의 모든 시트**를 조사합니다.

## 실행

```bash
cd /path/to/CNE_DNI
source .venv/bin/activate

# 전체 조사 (엑셀 5600+ 개 → 수 시간 소요 가능)
python -m src.survey_sheets

# 출력 경로 지정
python -m src.survey_sheets /path/to/충남 ./output

# 테스트: 엑셀 100개만 조사
python -m src.survey_sheets /path/to/충남 ./output 100
```

## 출력 파일 (output 폴더)

| 파일 | 설명 |
|------|------|
| 사전조사_폴더목록.csv | 모든 폴더 경로, 빈 폴더 여부, 파일 수, xlsx 수 |
| 사전조사_빈폴더목록.csv | 빈 폴더만 |
| 사전조사_엑셀없는_폴더목록.csv | 파일은 있으나 xlsx가 없는 폴더 |
| 사전조사_파일별_모든시트.csv | 파일 경로, 시트명, 행 수, 열 수, 1행 샘플 |
| 사전조사_요약.txt | 폴더 수, 빈 폴더 수, 엑셀 파일 수, 시트 레코드 수 |

## 참고

- **충남 전체** = 719개 학교. 자료는 통합 엑셀과 학교별 폴더에 나뉘어 있을 수 있음.
- **빈 폴더**가 있을 수 있으므로, 조사 결과로 누락 구간을 확인한 뒤 통합 로직을 보완하면 됨.
- 전체 조사는 엑셀 파일당 약 2~3초 걸릴 수 있어, 5600+ 파일 기준 **수 시간** 걸릴 수 있음. 백그라운드 또는 야간 실행 권장.
